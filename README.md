# 📊 Diabetes Prediction using Linear Regression (Sklearn vs From Scratch)

This project demonstrates the implementation of Linear Regression on the Diabetes dataset using two approaches:

1. 🔹 Using Scikit-learn  
2. 🔹 Custom implementation (from scratch)  

---

## 🚀 Objective

To predict disease progression based on medical features and compare the performance of a library-based model with a manually implemented model.

---

## 📁 Dataset

- Source: Scikit-learn Diabetes Dataset  
- Features: Age, BMI, Blood Pressure, etc.  
- Target: Disease progression score  

---

## ⚙️ Technologies Used

- Python  
- NumPy  
- Pandas  
- Scikit-learn  

---

## 🧠 Approach

### 🔹 Sklearn Model
- Used LinearRegression from sklearn  
- Trained on training data  
- Generated predictions on test data  

### 🔹 Custom Model
- Implemented Linear Regression manually  
- Calculated coefficients using mathematical formulas  
- Built a custom class for training and prediction  

---

## 🔧 Steps Performed

1. Loaded dataset using sklearn  
2. Train-test split  
3. Model training (sklearn + custom)  
4. Predictions on test data  
5. Comparison of results  

---

## 📈 Results

Both models produced similar predictions, validating the correctness of the custom implementation.

---

## 📌 Key Learnings

- Deep understanding of Linear Regression  
- Difference between manual implementation and library usage  
- Working with real-world datasets  
- Debugging and handling data correctly
